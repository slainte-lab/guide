# 🇧🇪 Брюгге — Аудиогид

Аудиогид по Брюгге на русском языке. Пешеходный маршрут на один день (13:00–22:00), 10 остановок, ~7 км.

## Что умеет

- **Аудио на русском** — синтез речи (Web Speech API), кнопка ▶ Слушать
- **Карта маршрута** — OpenStreetMap (CartoDB Dark), без API-ключей
- **Прогресс** — отмечайте посещённые точки, прогресс-бар вверху
- **Навигация** — кнопки ◀ ▶ между точками
- **Google Maps** — кнопка 📍 открывает текущую точку в картах
- **GPS-режим** — в коде есть готовый модуль (раскомментировать в `js/app.js`)

## Как запустить

### Локально
Просто откройте `index.html` в Chrome или Safari.

> ⚠️ На некоторых браузерах синтез речи не работает при открытии файла напрямую (`file://`). В таком случае запустите локальный сервер:
> ```bash
> npx serve .
> # или
> python3 -m http.server 8080
> ```

### GitHub Pages
1. Залейте репозиторий на GitHub
2. Settings → Pages → Source: `main` branch, `/ (root)`
3. Через минуту доступно по адресу `https://ваш-юзер.github.io/bruges-guide/`

## GPS-режим (автоматический)

В `js/app.js` в самом низу есть готовый код GPS-гида. Когда вы подходите к точке на 50 метров — гид автоматически запускает аудио. Чтобы включить:

1. Раскомментируйте `startGPS()` в конце `app.js`
2. Раскомментируйте функции `startGPS` и `getDistance`
3. Браузер попросит разрешение на геолокацию — разрешите

## Добавить новый город

Скопируйте `data/stops.js`, замените данные — и у вас новый гид. Карта и аудио подхватятся автоматически.

## Технологии

- [Leaflet.js](https://leafletjs.com/) — карта
- [CartoDB Dark](https://carto.com/basemaps/) — тёмные тайлы карты
- [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/SpeechSynthesis) — синтез речи
- Нет бэкенда, нет API-ключей, нет зависимостей npm
